ตั้งค่า extension — ทุกค่าจะถูกเก็บใน Chrome sync storage ของคุณเอง
ใช้เมื่อ extension หาเลขเครดิตอัตโนมัติไม่เจอ — ลอง inspect element บน freepik.com แล้ววาง CSS selector ของ element ที่มีเลขเครดิต
ตัวอย่าง: [data-testid="user-credits"] หรือ .header-credits .number
เปิดแท็บ freepik.com แล้วกดปุ่มนี้เพื่อทดสอบ
ตำแหน่งของ FastAPI backend — ใส่เป็น http://localhost:8765 ถ้ารันในเครื่องตัวเอง หรือ URL ของ Railway/server ถ้า deploy ไว้บน cloud
จำเป็นเฉพาะเมื่อ backend deploy บน internet — ดู key ได้ที่ Admin Console → 🔑 API Key
ถ้ารัน backend ในเครื่อง localhost ของตัวเอง ไม่จำเป็นต้องใส่ก็ใช้ได้ (แต่ถ้าใส่ก็จะใช้งานได้เหมือนกัน)
